Welcome to the Guessing game !

The objective of the guessing game is that the program will pick a random number between 0 and 99 and then you, the user will have to guess that number.

Do not worry though, the program will give you some hints if you are too loo or too high .

0 - 99

CLUE >>

Logic Explanation for Guessing Game

  • First ask the user for their guess.
  • When you get the Users, guess then use conditional statements to decide what to do with the input
  • if the guess < random number then tell the user the number is too low (output)
  • if the guess > random number then tell the user that the number is too high (output)
  • if the guess == random number then tell the user that the guess is right and end the game

Python Code for Guessing game


                        import random

                        n = random.randint(1, 99)
                        guess = int(raw_input("Enter an integer from 1 to 99: "))

                        while n != "guess":
                            print
                            if guess < n:
                                print "guess is low"
                                guess = int(raw_input("Enter an integer from 1 to 99: "))
                            elif guess > n:
                                print "guess is high"
                                guess = int(raw_input("Enter an integer from 1 to 99: "))
                            else:
                                print "you guessed it!"
                                break
                            print
