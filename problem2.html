Logic Explanation for Prime Number validator

  • First ask the user for the number they want to check.
  • Then check to make sure that the user gave you a postive number
  • if the user gave you a positive number then iterate from 2 to that number
  • In each iteration divide the number given by the user by the iteration position
  • if there is a number that divides the given input then we know that the given number is not a prime number
  • if there is no number that divides the given input then we know that the given number is a prime number

                        # Python program to check if the input number is prime or not
                            num = 407

                            num = int(input("Enter a number: "))

                            if num > 1:
                               # check for factors
                               for i in range(2,num):

                                       print(i,"times",num//i,"is",num)

                                   print(num,"is a prime number")

                            # or equal to 1, it is not prime
                            else:
                               print(num,"is not a prime number")
